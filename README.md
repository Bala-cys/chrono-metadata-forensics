## Overview
Chrono is a Python-based metadata extraction tool designed for analyzing file properties across multiple formats. It assists in security investigations and forensic analysis.

## Features
- Extracts metadata from images, documents, audio, and video files
- Generates file hashes (MD5, SHA256)
- Structured reporting
- Cross-platform compatibility

## Technologies Used
- Python
- Flask
- Hashlib
- Exif libraries

## Usage
Install dependencies:
pip install -r requirements.txt

Run:
python app.py
